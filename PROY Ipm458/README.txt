
En la carpeta se incluyen los siguientes directorios:

    - code/: Contiene lo siguiente:
        - main.f90: Código elaborado.
        - main.exe: Ejecutable.
        - FEM_Beckers.m: Código publicación Beckers y Beckers.
        - plots.py: Es un extracto del código de jupyter para graficar y leearlo más fácil.
        - results/: Contiene archivos con los resultados de la modelación.
        - eu_rtxt/ y cn_rtxt/: En el código se puede habilitar la opción de escribir cada cierto tout la distribucion de temperatura para cada esquema.
        - compiler.sh: Simple código shell para compilar y correr más fácil.
        - plot_code/: Se encuentra plots.ipynb, código en python jupyter notebook para crear las gráficas y un .html para poder leerlo si lo requiere.

    - Plots/: Contiene lo siguiente:
        - plots/: Contiene las gráficas presentadas en el informe para todas las modelaciones.
        - plots_evolution/: Contiene las temperaturas para cada paso de salida tout.
        - videos/: Contiene los gifs para las modelaciones.

    Además se incluye el informe elaborado y el archivo README.
